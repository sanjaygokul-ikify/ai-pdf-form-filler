# Contribution Guidelines
## Introduction
The ai-pdf-form-filler library is an open-source project that relies on contributions from the community. We welcome contributions in the form of bug reports, feature requests, and code submissions.
## Code Style
The ai-pdf-form-filler library uses the PEP 8 coding standard. Please ensure that your code conforms to this standard.
## Commit Messages
Commit messages should follow the GitHub guidelines. Use the imperative mood and keep the message concise.
## PR Workflow
To contribute code, fork the repository and submit a pull request. The pull request should include a clear description of the changes and a reference to the issue being addressed.
## Issue Tracking
Issues are tracked using the GitHub issue tracker. To report a bug or request a feature, create a new issue and provide a clear description of the problem or request.
